# String-Finder

GDD JSON 파일에서 문자열을 빠르게 검색하는 Windows 애플리케이션입니다.

## 개발 목적

string 작업 시 다음과 같은 불편을 해소하기 위해 개발했습니다.

- **오버플로우 감지**: 번역 결과를 표시 너비순으로 정렬하여 문제가 될만한 케이스를 빠르게 확인할 수 있습니다.
- **유의어 확인**: 입력한 검색어의 유의어가 GDD에 등록되어 있는지 확인할 수 있습니다.
- **예상 번역 확인**: 번역이 완료되지 않은 string의 경우에도 예상 번역을 확인할 수 있으며, 예상 번역이 긴 경우 축약을 요청할 수 있습니다.
- **빠른 검색**: 엑셀을 켜고 적합한 파일을 여는 데 드는 시간을 크게 줄였습니다.
- **유연한 검색**: 카테고리 내에서만 가능한 기존 검색 구조를 변경하여 통합 검색을 지원합니다.
- **데이터 최신화**: Git Pull 기능으로 소스트리와 엑셀 프로그램으로 이원화된 작업을 통합했습니다.

## 시스템 요구사항

- Windows 10 이상 (64bit)
- Git 설치 (Pull 기능 사용 시)

## 이용 가이드

### 설치

- **Installer**: `String-Finder-{version}-x64.exe` 파일을 다운로드하여 설치합니다.
- **Portable**: `String-Finder-{version}-portable.exe` 파일을 다운로드하여 설치 없이 바로 실행할 수 있습니다.

### 초기 설정 (필수)

1. **경로 설정**: 프로그램 실행 후 우측 상단의 ⚙️ 버튼을 클릭하여 `game-design-data/localization/ui` 폴더 경로를 지정해야 합니다.
   - 올바른 경로가 설정되지 않으면 검색 기능을 사용할 수 없습니다.
2. **데이터 동기화**: 🔄 버튼을 클릭하여 최신 데이터를 받아옵니다 (`git pull`).

### 검색 기능

**검색 모드**
- **GDD**: GDD 데이터 내에서 String ID 또는 텍스트를 검색합니다.
- **유의어**: 검색어의 유의어를 찾아 해당 유의어가 포함된 String을 검색합니다.
- **AI 번역**: 검색어에 대한 AI 예상 번역을 생성합니다.

**검색 방법**
1. 언어를 선택합니다 (예: KO).
2. 검색어를 입력하고 Enter 키를 누르거나 검색 버튼을 클릭합니다.
3. 최근 검색 기록이 저장되어 나중에 다시 선택할 수 있습니다.

### 검색 결과 활용

- **상세 보기**: 검색 결과 항목을 클릭하면 해당 String ID의 모든 언어별 번역을 확인할 수 있습니다.
- **복사**: 📋 버튼을 눌러 텍스트를 클립보드에 복사할 수 있습니다.

### AI 번역 활용

- **표준 번역**: AI가 예측한 기본 번역을 확인합니다.
- **축약 번역**: '축약 번역' 탭을 선택하면, 표시 너비가 긴 상위 10개 언어에 대해 축약된 번역을 제안받을 수 있습니다.

### 기타 기능

- **패치 노트**: 🗒️ 버튼을 눌러 업데이트 내역을 확인합니다.
- **가이드**: ❓ 버튼을 눌러 상세 가이드 페이지(Confluence)로 이동합니다.
- **상태 표시**: 하단 상태바에서 마지막 데이터 업데이트 시간을 확인할 수 있습니다.
